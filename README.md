## 230823 ##
1. 로컬 저장소 생성<br>
`$ git init`

2. README.md 파일 생성<br>
`$ git add .`<br>
`$ git commit -m "README.md 파일 생성"`

3. 온라인 저장소 생성<br>
`$ git remote add origin https://repo주소.git`<br>

4. README.md 파일 온라인 저장소에 업로드<br>
`$ git push -u origin main`

### github에서 내려 받기 ###
`$ git pull origin main`
